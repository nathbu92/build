Connexion à Pronote

Connecte-toi avec ton compte Pronote pour accéder à tes données scolaires.

Comment obtenir le QR Code ?
Dans l'appli Pronote mobile : Paramètres → Mon compte → Connexion avec QR Code.
Scanne-le avec ton téléphone et note le code PIN à 4 chiffres.

🔒 Tes identifiants ne quittent jamais ton appareil.
Connexion directe au serveur Pronote de ton lycée.
